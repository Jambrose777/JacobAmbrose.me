SoutheastCubing.org
SoutheastCubing.org is a website made to collect all information about Cubing Competitions for Southeast Cubers in one place. The website provides descriptions of our Delegates, a list of upcoming competitions, and information on local clubs. Here is a link to the codebase.
Specs
This website was made very similarly to JacobAmbrose.me. It has an Angular frontend, uses AWS (EC2, S3, and Route 53). Apache to run the web server.
CMS
All content on the website is hosted using a Content Management System (CMS), Contenful. Many of the Delegates who are stakeholders for the website have little to no coding experience, so they are able to use the CMS to modify any content.
