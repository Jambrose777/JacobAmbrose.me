Saia - Website Redesign

July 2018 - July 2019
On this project, I revamped Saia.com and its contributing systems. We were tasked to replace, the old website to make a modern look and feel as well as improve their severe preformance issues when making trasnactions with their backend. This required learning about Visual Basic (VB) code to learn how their current systems worked.

Frontend

The new website uses an Angular frontend. We developed the site as a reactive web app, so that users can use any device to receive content on the website. Saia has several pages for forums (ie Obtaining Quotes) that we revamped to have an improved user experience. We implemented Google Analytics to visually provide the buisness with several dashboard to show site productivity.
Saia has several pages for general content. Previously, buisness stakeholders would have to learn html and provide developers with what they want for each page, including when they wanted changes to the website. We utilized a headless content management system (CMS) to create an easy way for buisness to edit and manage their content on the website. They no longer have to learn coding nor rely on coding deployments, they can instead log onto the CMS, find the content they want to edit, and edit it with a markdown editor.

Backend

We developed several backend micro services. Each micro service was responsible for a specific section of the site (ie Quotes). If the site was experiencing a high work load on the Quotes pages, then the microservice for Quotes would scale up to the demand without affecting any other micro service.
These new services have improved logic as well as optimized database interactions. We utilized asynchronise calls to improve preformance relying on the database. This lowered the turnaround time (ie for getting a Quote) from over a minute to under 5 seconds.
