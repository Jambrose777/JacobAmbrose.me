Pulte - Home Designer (PHD)

March 2021 - July 2022
For this project, I created a website to aid home buyers in finding and selecting attributes for a home that they are interested in. The application allows users to favorite items that would potentially be contracted for their home at a later time. I worked as a Developer Team Leader on this project. I spearheaded many software solutions and onboarded new team members.

Frontend

The website uses an Angular frontend. We heavily utilized a local storage system, NgRx to store all neccessary information for reloading the application without needing to make repeated API calls. This project uses Aphavision to display different home layouts and have an interactive user experience.

Branding / Theming

Branding is where a website changes it's theme based on which company the website is being used for. This is similar to having a light-mode vs dark-mode on a website, but we may also change images and sometimes layouts. I developed a system for this project to determine the correct brand and display the website with the correct theme. This was done using styling variables and allowed for development of only 1 codebase for 6 different brands / applications.

Backend / Security

This project used a C# backend. I worked heavily on the security of the API determining various roles and permissions for different user types. I ensured that only authenticated users had access to certain API endpoints. Those endpoints also retrieved only information that the user is allowed to have access to.
